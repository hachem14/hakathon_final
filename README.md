# 💊 MedSafe - BioVatech Hackathon

**MedSafe** is a desktop application designed to combat self-medication risks in Algeria. It provides healthcare professionals (doctors and pharmacists) with tools to verify drug interactions and explain risks to patients using **local Darija explanations**.

## ✨ Key Features
* **Interaction Analyzer**: Instantly check compatibility between two drugs.
* **Doctor Dashboard**: Manage consultations and track patient prescriptions.
* **Pharmacist Space**: Validate prescriptions and provide patient-friendly advice.
* **Linguistic Accessibility**: Interaction results are explained in Darija for better patient understanding.

## 🛠️ Prerequisites
Make sure you have **Python 3.x** installed.
